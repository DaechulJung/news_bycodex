원문 상세 요약 - LangChain은 GLM-5와 MiniMax M2.7을 Deep Agents 하네스 평가에 넣은 결과, 파일 작업, 도구 사용, instruction following에서 closed frontier 모델과 함께 쓸 수 있는 수준이라고 설명합니다. - 평가 범주는 file operations, tool use, retrieval, conversation, memory, summarization, unit tests의 7개이며 correctness, solve rate, step ratio, tool call ratio를 함께 봅니다. - 공개 표에서 GLM-5는 138개 중 94개 통과, correctness 0.64를 기록했고 MiniMax M2.7은 85개 통과, correctness 0.57을 기록했습니다. - 비교군으로 Claude Opus 4.6은 100/138, Gemini 3.1 Pro Preview는 96/138, GPT-5.4는 91/138로 제시되었습니다. - 비용 예시는 Claude Opus 4.6 output $25/M tokens, GPT-5.4 output $15/M tokens, GLM-5 output $3.15/M tokens, MiniMax M2.7 output $1.20/M tokens로 제시됩니다. - Deep Agents SDK에서는 model 문자열을 바꾸는 방식으로 GLM-5 또는 MiniMax M2.7을 사용할 수 있고, Baseten, Fireworks, Groq, OpenRouter, Ollama cloud 같은 provider가 언급됩니다. - Deep Agents CLI에는 런타임 모델 스와핑을 위한 middleware와 /model slash command가 소개되며, frontier 모델로 계획하고 open 모델로 실행하는 패턴을 제안합니다. 인사이트 - agent harness는 모델 라우팅, provider abstraction, 비용 최적화, tool-call 호환성을 흡수하는 계층이 되고 있습니다. 확인 포인트 - Deep Agents GitHub stars/forks/release cadence, eval 재현 스크립트, CI 결과, provider별 latency와 context limit 차이를 후속 수집해야 합니다.

Delta Channels: How We’re Evolving our Runtime for Long-Running Agents

Delta Channels는 LangGraph 1.2의 diff 기반 checkpoint primitive로, 장기 실행 에이전트의 full-state checkpoint O(N²) 저장 문제를 줄입니다.
원문 상세 요약 - 장기 실행 에이전트는 매 step마다 full state를 checkpoint하면 저장량이 O(N²)로 증가하는 문제가 있습니다. - DeltaChannel은 LangGraph 1.2의 새 primitive입니다. - 각 step에서 전체 상태가 아니라 diff만 checkpoint합니다. - 주기적으로 full snapshot을 기록해 복구 기준점을 유지합니다. - 목표는 세션이 길어져도 storage costs를 flat하게 유지하는 것입니다. - Deep Agents v0.6에 기본 탑재되며 config 변경이나 data migration이 필요 없다고 입력에 명시되어 있습니다. 인사이트 - production agent harness의 병목이 모델 호출 비용뿐 아니라 상태 저장 구조와 복구 비용으로 이동하고 있음을 보여줍니다. 확인 포인트 - diff checkpoint가 replay, 감사 로그, partial rollback, concurrent tool execution에서 어떤 제약을 갖는지 확인해야 합니다.

Delta Channels: How We’re Evolving our Runtime for Long-Running Agents

#agent_framework #harness_engineering #runtime #memory #orchestration #cost #release #tooling released high 신호 4/5
LangGraph 1.2 DeltaChannel은 long-running agents의 full-state checkpoint O(N²) 저장 문제를 diff checkpoint와 주기적 snapshot으로 줄이는 런타임 primitive입니다.
원문 상세 요약 - 입력은 long-running agents가 checkpoint storage problem을 갖는다고 설명합니다. - 매 step마다 full state를 저장하면 세션이 길어질수록 저장 비용이 O(N²)로 증가합니다. - DeltaChannel은 LangGraph 1.2의 새 primitive입니다. - 각 step에서는 전체 상태 대신 diff만 checkpoint합니다. - full snapshot은 주기적으로 기록합니다. - 세션이 길어져도 storage costs를 flat하게 유지하는 것이 목표입니다. - Deep Agents v0.6에는 기본 포함되며 config 변경이나 data migration 없이 제공된다고 설명됩니다. 인사이트 - 장기 실행 에이전트의 실제 비용은 토큰뿐 아니라 state replay, checkpoint storage, resume reliability에서 발생합니다. DeltaChannel은 운영비와 장애 복구 설계를 직접 겨냥합니다. 확인 포인트 - backend별 비용 절감 폭, snapshot 주기 설정, rollback과 audit replay 신뢰성, custom channel 호환성을 검증해야 합니다.

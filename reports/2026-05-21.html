#langchain #langgraph #deep_agents #delta_channel #checkpointing #state_management #long_running_agents released high 신호 4/5

원문 상세 요약 - LangChain은 long-running agents가 매 step마다 full state를 checkpoint하면 저장 비용이 O(N²)로 커진다고 설명합니다. - DeltaChannel은 LangGraph 1.2의 새 primitive로 각 step에서 전체 상태가 아니라 diff만 checkpoint합니다. - 전체 snapshot은 주기적으로 기록해 긴 세션에서도 storage cost가 flat하게 유지되도록 설계됐습니다. - Deep Agents v0.6에 기본 탑재됩니다. - 입력 요약에는 no config changes, no data migration required가 명시되어 있습니다. 인사이트 - 장기 실행 에이전트의 병목은 모델 비용뿐 아니라 checkpoint 저장량, 복구 시간, 상태 재생 비용으로 확장되고 있습니다. - agent harness는 memory/state store를 단순 로그 저장소가 아니라 증분 상태 관리 계층으로 설계해야 합니다. 확인 포인트 - snapshot 주기, 실패 복구, diff schema, 민감정보 redaction/retention 적용 방식은 후속 확인이 필요합니다.
